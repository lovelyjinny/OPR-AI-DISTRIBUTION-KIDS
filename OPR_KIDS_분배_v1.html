매장 관리 · 15개 매장
진열량과 등급(1~15위)을 직접 입력하세요. 매입·직납 공통으로 사용됩니다.
진열량 입력 · 행거 개수 × 행거당 수량(어바웃)으로 입력. 직납의 적정재고 계산에 사용됩니다.

직납 PO 파라미터
발주량 = 매출×비율 − (현재고+미확정PO)
이 미만은 발주 안 함
필터
1K (아동) MCH2만 분배
OFA(아동_OPR) 구매그룹만 분배
자라키즈 출고 포함기본 제외 — 이벤트성 출고 시에만 ON
도움말
아동상품 2가지 형태
매입 OPR이 매입 → 물류센터 보유 → STO 분배 (분배장 출력)
직납 공급업체가 매장 직접 배송 → 구매오더(PO) 생성
직납 브랜드 6개: 올리반 · 디즈니캐릭터 · 해피버스 · 쁘띠뮤 · 삠뽀요 · 팬콧
매입 픽업 로직
  1. 매출상세에서 매장×자재별 판매수량 추출
  2. 가용재고에 동일 자재 있으면 보충 (판매분 × 픽업비율 × 매장가중치)
  3. 동일 자재 부족 → 사이즈 대체 (자재명 prefix 매칭)
  4. 그래도 부족 → 미배정 기록
직납 PO 로직
  1. 현시점 매장재고 + 미확정 PO = 현재 보유량
  2. 발주량 = 매출수량 × 발주비율 − 현재 보유량
  3. 발주량 > 0 인 경우만 PO 발주
  4. 공급업체(브랜드)별 시트로 분리 출력
